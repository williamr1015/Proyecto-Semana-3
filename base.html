Inicios

Bootstrap fue diseñador por Mark Otto y Jacob Thornton ambos en Twitter crearon este Framework para usarlo internamente en los proyectos de la compañía.

Bootstrap 1

Después de unos meses de desarrollo, se renombró de Twitter Blueprint a Bootstrap y se lanzó como proyecto de código abierto el 19 de agosto de 2011.

Bootstrap 2

Mark Otto anunció Bootstrap 4 el 29 de octubre de 2014. La primera versión alfa de Bootstrap 4 se lanzó el 19 de agosto de 2015​. La primera versión beta se lanzó el 10 de agosto de 2017.

Bootstrap 3

Bootstrap 3, fue lanzada el 19 de agosto de 2013. Rediseñó los componentes para usar un diseño plano y un primer enfoque móvil.

Bootstrap 4

Bootstrap fue diseñador por Mark Otto y Jacob Thornton ambos en Twitter crearon este Framework para usarlo internamente en los proyectos de la compañía.

Bootstrap 5

La versión 5 de Bootstrap fue anunciada por Mark Otto el 21 de diciembre de 2018 y, luego de 3 versiones alfas y 3 betas, fue lanzada oficialmente el 5 de mayo de 2021.

Actualidad

Bootstrap 5 no ha reinventado toda su estructura. Esto concuerda con la intención de sus desarrolladores, que pretendían hacer una transición fácil de la versión 4 a la 5.

Usos

Bootstrap ofrece una serie de características que se pueden implementar en un sitio web, estas son Diseño Responsive y Bibliotecas de Componentes.
